## Tips for Post-Processing
- Prefer Simulator/device captures for Screenshots 2–6 when possible; use AI only for art direction beds or title splash variants
- Add a subtle device frame (iPhone 15 Pro landscape) only if App Store layout needs it
- Place callout text at the top or bottom in a semi-transparent banner — not on top of critical HUD
- Overlay fonts: SF Rounded Bold / system rounded to match in-game UI
- Keep HUD colors exact: budget **green**, XP **cyan**, LURE **orange**, endless timer **purple**
- Exact upgrade card titles if overlaying Screenshot 4: `Sneaker Speed`, `New: Barcode Laser`, `Coupon Printer`
- Exact result copy if overlaying Screenshot 6: `BUDGET SURVIVED` / `Unlocked: Midnight Mall (Endless)`
