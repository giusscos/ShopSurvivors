## Tips for Post-Processing
- Add a subtle device frame (iPhone 15 Pro in landscape) around each image
- Place callout text at the top or bottom in a semi-transparent pill/banner
- Use font: SF Rounded Bold or any rounded sans-serif for consistency with the game's UI
- Maintain consistent color palette: **cyan #33D9E8**, **orange #FF8C40**, **dark navy #1A2A36**
